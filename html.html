HTML — The Structure of Every Web Page

HyperText Markup Language is the foundation of the web. Every website you've ever visited is built on HTML. It tells the browser what content to show and how to organise it — headings, paragraphs, images, links, and more.

What is HTML?

How HTML Works

HTML uses "tags" wrapped in angle brackets to mark up content. The browser reads these tags and renders them into the visual page you see.

1

You Write Tags

Tags like <h1> tell the browser "this is a heading". Most tags come in pairs — an opening tag and a closing tag.

2

Browser Parses It

The browser reads your HTML file from top to bottom and builds a tree structure in memory called the DOM (Document Object Model).

3

Page is Rendered

The browser uses the DOM to paint pixels on screen — turning your text tags into visible headings, paragraphs, images and links.

4

CSS & JS Take Over

HTML provides the structure. CSS makes it beautiful and JavaScript makes it interactive. Together they build every website you visit.

Examples

What HTML Can Build

HTML is capable of creating a huge variety of content and structures, all without any CSS or JavaScript.

Text & Headings

h1–h6 for headings, p for paragraphs, strong, em, blockquote — rich typographic content from pure HTML.

Links & Navigation

The <a> tag creates hyperlinks — the very foundation of the web. Link to pages, sections, emails, or files.

Images & Media

Embed images with <img>, videos with <video>, audio with <audio>, and iframes for external content.

Tables

Structured data in rows and columns using <table>, <tr>, <td>, and <th> — perfect for schedules and reports.

Forms & Inputs

Collect user data with <form>, text inputs, dropdowns, checkboxes, radio buttons, and submit buttons.

Semantic Layout

Use <header>, <nav>, <main>, <article>, <section>, <footer> for meaningful, accessible page structure.

HTML Form Example

This is a real HTML form rendered live in your browser — no CSS or JavaScript involved.

contact-form.html
<form>
  <label for="name">Your Name</label>
  <input type="text" id="name" placeholder="John Doe">

  <label for="email">Email Address</label>
  <input type="email" id="email" placeholder="john@example.com">

  <label for="msg">Message</label>
  <textarea id="msg" rows="3"></textarea>

  <button type="submit">Send Message</button>
</form>
