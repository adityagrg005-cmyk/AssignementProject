What is CSS?

How CSS Works

CSS selects HTML elements and applies styles to them. You write rules in the format selector { property: value; }

Selectors

Target elements with tag names (p), class names (.card), IDs (#header), or complex combinators.

Properties

Over 200 CSS properties control color, size, spacing, font, border, shadow, layout, animation and more.

Cascade

The "C" in CSS — rules cascade from general to specific. More specific selectors win when styles conflict.

Responsive

Media queries let you write different styles for different screen sizes — the key to mobile-friendly design.

Flexbox & Grid

Modern layout systems that make complex, responsive page layouts simple and powerful.

Animations

Bring pages to life with transitions, keyframe animations, and transform effects — all with pure CSS.
