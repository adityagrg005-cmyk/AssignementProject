Core Concepts

What You Need to Know

These are the essential building blocks of JavaScript that every developer uses every day.

Variables

Store and reuse values with let, const, and var. Variables can hold numbers, strings, arrays, objects, and more.

Functions

Reusable blocks of code. Call them whenever needed. Arrow functions () => {} are modern shorthand.

Arrays

Ordered lists of values. Use .map(), .filter(), .forEach() to process them.

Objects

Key-value pairs that represent real-world things. { name: "JS", year: 1995 }

DOM Events

Listen for user actions — clicks, keypresses, form submissions — and respond with addEventListener.

Conditions

Control flow with if/else, ternary operators, and switch statements.

// Arrow function
const calcGrade = (score) => {
  if (score >= 90) return 'A';
  if (score >= 75) return 'B';
  if (score >= 60) return 'C';
  return 'F';
};

[95, 80, 65, 45].forEach(s =>
  console.log(`${s} → Grade ${calcGrade(s)}`)

arrays.js
const scores = [85,92,78,95,61,88];

const avg = scores.reduce((a,b) => a+b,0)
           / scores.length;

const passing = scores.filter(s => s >= 75);

console.log(`Average: ${avg.toFixed(1)}`);
console.log(`Passing: ${passing.length}/6`);
console.log(`Top: ${Math.max(...scores)}`);
