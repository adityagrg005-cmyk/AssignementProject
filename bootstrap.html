What is Bootstrap?

Why Developers Love Bootstrap

Bootstrap removes the need to write repetitive CSS from scratch. Just add class names to your HTML and get professional results instantly.

12-Column Grid

Bootstrap's grid system divides the page into 12 columns. Use col-md-6 for half-width, col-md-4 for thirds.

Mobile First

Breakpoints: sm (576px), md (768px), lg (992px), xl (1200px). Designs adapt to any screen.

Components

Ready-made navbars, cards, modals, dropdowns, alerts, badges, carousels — all styled and accessible out of the box.

Utility Classes

Hundreds of utility classes for spacing (p-3, m-2), text (text-center), color, display and more.

Fast Prototyping

Build a fully responsive multi-page website in hours instead of days. Perfect for projects, portfolios, and client demos.

Customisable

Override Bootstrap's default variables with your own colors, fonts, and spacing using CSS custom properties or Sass.
